WTikZ Embedded Drawing Example

This page demonstrates how to embed a .wtikz drawing inside any web page using the wtikz-viewer.js library.

1. Link the library

<script src="wtikz-viewer.js"></script>

2. Prepare a container

<div id="drawing-container"></div>

🚀 Best Way: Custom HTML Tag (One-liner)

Just add this one script and use the tag anywhere in your HTML!

<script src="wtikz-viewer.js"></script>
<wtikz-image src="my_drawing.wtikz" width="600px" height="300px"></wtikz-image>
