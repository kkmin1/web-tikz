# **TikZ Studio(WYSIWYG 에디터) 사용법**

이 에디터는 시각적으로 드로잉한 요소를 LaTeX TikZ 코드로 실시간 변환해주는 웹 기반 도구입니다.

## **1. 구조 모듈화**

코드 베이스가 확장됨에 따라 구조가 깔끔하게 분리되어 관리됩니다:
* `web-tikz.html`: 메인 레이아웃 및 UI 컨테이너
* `style.css`: 각종 속성 패널, 캔버스 동작 애니메이션, 커스텀 스타일 속성
* `app.js`: 핵심 로직 (그리기, 상태 관리, TikZ 모수 변환 엔진 등)

## **2. 기본 도구 설명**

상단 툴바의 아이콘을 클릭하여 요소를 캔버스에 추가하거나 조작할 수 있습니다.
* **Select (마우스)**: 요소를 선택, 이동, 크기 조절하거나 우측 패널에서 속성을 변경합니다.  
* **Circle / Rectangle (원 / 사각형)**: 노드를 생성합니다. 
* **Line / Curves (선 / 곡선 도구)**: 두 점을 연결하는 직선이나 곡선을 생성합니다.
  * **Multiple Arrows (다중 화살표)**: Properties 패널에서 `Arrow` 체크박스와 `Count`를 통해 선 위에 여러 화살표를 일정한 간격으로 배치할 수 있습니다.
  * **Direction Modes (방향 모드)**: 화살표의 진행 패턴을 결정합니다.
    * `Forward` / `Backward`: 항상 선의 끝점(Forward) 또는 시작점(Backward)을 향합니다. (이때 Flip Points의 영향을 받지 않습니다.)
    * `Inward` / `Outward`: 설정한 '반전 지점(Flip Points)'을 기준으로 화살표들이 모여들거나(수렴) 퍼져나가는(발산) 형태로 그려집니다.
  * **Flip Points (반전 지점 설정)**: 화살표의 방향이 꺾이는 지점을 0.0과 1.0(시작과 끝) 사이의 비율로 지정합니다. 콤마(`,`)로 구분하여 여러 개를 입력할 수 있습니다. (예: `0.3, 0.7`)
  * 이 기능은 TikZ의 `decorations.markings` 라이브러리를 사용하여 구현됩니다.
* **Bezier Curves (곡선 도구 류)**
  * **Cubic (기본 베지어, 컨트롤 포인트 2개)**: 4점 기반의 매끄러운 3차 곡선을 그립니다.
  * **Quadratic (무지개, 컨트롤 포인트 1개)**: 3점 기반의 2차 곡선을 그립니다.
  * **Quartic (파도, 컨트롤 포인트 3개)**: 5점 기반의 복잡한 4차 곡선(TikZ 모수 방정식 변환 지원)을 그립니다.
* **Function Plot (그래프 기호)**: 수학 함수 $y=f(x)$를 기준 영역 위로 정확히 렌더링하고 `\draw plot` 표준 문법으로 추출합니다. 수식은 js 수식 문법을 따른다. 예) 2*x, -(x**2), Math.exp(x)
* **Eraser (지우개)**: 드래그한 영역에 걸친 모든 요소를 일괄 삭제합니다.

## **3. 수식 기능 및 그래프 조작**

**1) 일반 노드 내 수식 입력**
오른쪽 Properties 패널의 Text 속성에서 LaTeX 문법을 `$E=mc^2$` 처럼 `$` 기호로 감싸면 캔버스 자체에서 MathJax로 실시간 렌더링 뷰를 제공합니다.

**2) Function Plot 속성 제어**
함수 도구(Plot)로 생성된 객체를 클릭하면, 우측 패널에서 함수식을 정의할 수 있습니다.
* **Preview Math (JS)**: 자바스크립트 엔진으로 브라우저 캔버스 위에 실시간 렌더링(`Math.sin(x) * x` 등)
* **Export Math (TikZ)**: TikZ 문서용 모수 방정식(`sin(\x r) * \x`)
* **Domain / Samples**: 정의역 시작과 끝, 부드러움을 결정할 샘플링 정밀도

## **4. 단축키 및 상태 관리 (Undo / Redo)**

단축키 지원을 통해 작업 속도가 향상됩니다:
* `Ctrl + Z`: 되돌리기 (Undo)
* `Ctrl + Y` 또는 `Ctrl + Shift + Z`: 다시 실행 (Redo)
* `Delete` 또는 `Backspace`: 선택한 요소 즉시 삭제
  *주의: 패널 속성값을 편집 중(인풋 박스 활성화)에는 단축키가 작동하지 않도록 보호됩니다.*

## **5. 데이터 관리 및 내보내기**

* **캔버스 크기 조절**: 상단 Size 입력칸에 가로, 세로 픽셀을 입력해 뷰포트를 조절합니다.  
* **Copy TikZ**: 클릭 즉시 하단에 생성된 모든 코드를 클립보드에 복사해 LaTeX 에디터에 붙여넣습니다.  
* **Save / Load**: 
  * 최신 **File System Access API**를 이용하여 윈도우/맥 OS 네이티브 경로 저장 다이얼로그가 열리며 폴더 및 파일명(`.json`)을 직접 지정해 백업 가능합니다. API 미지원 브라우저의 경우 파일명 프롬프트Fallback을 지원합니다.
