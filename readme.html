TikZ Studio 사용 설명서

이 에디터는 시각적으로 드로잉한 요소를 LaTeX TikZ 코드로 실시간 변환해주는 웹 기반 도구입니다.

1. 구조 모듈화

코드 베이스가 확장됨에 따라 구조가 깔끔하게 분리되어 관리됩니다:

  • web-tikz.html: 메인 레이아웃 및 UI 컨이동
  • style.css: 각종 속성 패널, 캔버스 동작 애니메이션, 커스텀 스타일 속성
  • app.js: 핵심 로직 (그리기, 상태 관리, TikZ 모수 변환 엔진 등)

2. 기본 도구 설명

상단 툴바의 아이콘을 클릭하여 요소를 캔버스에 추가하거나 조작할 수 있습니다.

  • Select (마우스): 요소를 선택, 이동, 크기 조절하거나 우측 패널에서 속성을 변경합니다.
  • Circle / Rectangle (원 / 사각형): 노드를 생성합니다.
  • Line / Curves (선 / 곡선 도구): 두 점을 연결하는 직선이나 곡선을 생성합니다.
    • Multiple Arrows (다중 화살표): Properties 패널에서 Arrow 체크박스와 Count 수식을 통해 선 내부 여러 곳에 화살표를 일정한 간격으로 배치합니다.
    • Direction Modes (방향 모드): 화살표의 진행 패턴을 결정합니다.
      • Forward / Backward: 항상 선의 끝점/시작점을 향합니다. (이때 Flip Points 설정은 무시됩니다.)
      • Inward / Outward: 지정한 '반전 지점(Flip Points)'을 기준으로 화살표들이 모여들거나(수렴) 퍼져나가는(발산) 형태로 그려집니다.
    • Flip Points (반전 지점 다중 설정): 화살표의 방향이 꺾이는 지점을 0.0과 1.0(시작과 끝) 사이의 비율로 지정합니다. 텍스트 창에 콤마(,)로 구분하여 여러 개 입력할 수 있습니다. (예: 0.3, 0.7)
    • 이 기능들은 TikZ의 decorations.markings 라이브러리를 사용합니다. 하단 TikZ 출력창의 주석을 참조하세요.
  • Bezier Curves (곡선 도구 류)
    • Cubic (기본 베지어, 컨트롤 포인트 2개): 4점 기반의 매끄러운 3차 곡선을 그립니다.
    • Quadratic (무지개, 컨트롤 포인트 1개): 3점 기반의 2차 곡선을 그립니다.
    • Quartic (파도, 컨트롤 포인트 3개): 5점 기반의 복잡한 4차 곡선을 그립니다.
  • Function Plot (그래프 기호): 수학 함수 $y=f(x)$를 정확히 렌더링하고 \draw plot 문법으로 추출합니다. 수식은 JS 문법을 따릅니다 (예: 2*x, -(x**2), Math.exp(x)).
  • Eraser (지우개): 드래그한 영역에 걸친 요소를 일괄 삭제합니다.

3. 수식 기능 및 그래프 조작

1) 일반 노드 내 수식 입력
Properties 패널의 Text 속성에서 LaTeX 문법을 $E=mc^2$ 처럼 $ 기호로 감싸면 실시간 렌더링됩니다.

2) Function Plot 속성 제어
함수 도구를 클릭하면 우측 패널에서 수식을 정의할 수 있습니다.

  • Preview Math (JS): 브라우저 렌더링용 (Math.sin(x) 등)
  • Export Math (TikZ): TikZ 문서 내보내기용 (sin(\x r) 등)
  • Domain / Samples: 영역 범위 및 정밀도 설정

4. 단축키 및 상태 관리

  • Ctrl + Z: 되돌리기 (Undo)
  • Ctrl + Y / Ctrl + Shift + Z: 다시 실행 (Redo)
  • Delete / Backspace: 선택 요소 삭제

5. 내보내기

  • Copy TikZ: 코드를 클립보드에 복사합니다.
  • Save / Load: JSON 파일로 작업을 저장하거나 불러옵니다.
